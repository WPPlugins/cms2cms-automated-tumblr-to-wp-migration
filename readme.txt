Automated Tumblr To WordPress Migration Plugin

== Description ==

You have been running your website on the Tumblr platform for some time already. But you’ve checked out a WordPress CMS and decided that WP would suit you better. Now you are wondering if and how you can move your valuable and extensive data from Tumblr to WordPress. Furthermore, you want this migration to be flawless and accurate, since you may be a busy person or have no or little coding skills.

= How does this plugin help: =
- Preview the prospective outcomes with the Free Demo Migration;
- The time of the migration depends on the content volume you’re moving from Tumblr to WordPress);
- Identify the content items you want to transfer from Tumblr to WordPress CMS (Data “Mapping” option);
- The price for complete migration depends on the content amount you want to migrate to WordPress.

= What you can move from Tumblr to WordPress: =
*(Note that CMS2CMS moves only content data from Tumblr to WordPress)*
* Pages along with the content included there;
* Images included in the content body;

= Preserved SEO rankings after the migration =
* Migration of keywords, metadata/meta description.
* 301 Redirects from Tumblr URLs to WordPress URLs

= Compatible with: WordPress 4.1.x - 4.8.x. =
- Live help is available. Our support team is here 24/7.
- When a custom migration is needed, CMS2CMS offers data migration service packages to order.

== Installation ==

1. Download the plugin zip file
1. Extract plugin zip file to your PC
1. Upload extracted file to wp-content/plugin directory
1. Go to Admin -> Plugins, find “CMS2CMS tumblr to WordPress Migration” and click Activate
1. Register an account
1. Setup Demo Migration
1. You’ll be redirected to CMS2CMS website in order to complete your migration
